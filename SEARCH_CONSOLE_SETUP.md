# Подключение к поисковикам — пошагово

После пуша этого коммита **сайт уже технически готов** (sitemap, JSON-LD, Product страницы, IndexNow пингует Bing/Yandex автоматически). Чтобы Google и Yandex реально показали сайт в выдаче — нужна **верификация владельца** (это требование поисковиков, нельзя обойти автоматически).

Каждое действие = ~3 минуты.

---

## 1. Google Search Console (самое важное)

1. Открой **https://search.google.com/search-console** → войди под Google-аккаунтом
2. **Добавить ресурс** → выбери **«URL prefix»** → введи `https://paparazzifashion.shop`
3. Google предложит метод верификации. Выбирай **«HTML file»**:
   - Скачай файл `googleXXXXXXXXXX.html`
   - Скажи мне его имя (например `google5a2b3c4d.html`) — я положу его в корень сайта и запушу
4. После пуша (~2 мин на деплой) — нажми «Verify» в Search Console
5. Когда верификация пройдёт:
   - Перейди в **Sitemaps** (левое меню)
   - Введи `sitemap.xml` → нажми **Submit**
   - Готово. Google начнёт индексацию (первые страницы появятся через 1–7 дней).

---

## 2. Yandex Webmaster (для русскоязычных клиентов)

1. Открой **https://webmaster.yandex.com** → войди (Yandex / Mail.ru аккаунт)
2. **Add site** → введи `https://paparazzifashion.shop`
3. Выбирай метод **«HTML file»** (как у Google)
   - Скачай файл вида `yandex_XXXXXXXXXX.html`
   - Скажи мне имя — добавлю в корень
4. После пуша → жми **«Check»** в Yandex Webmaster
5. После верификации → **Indexing → Sitemap files** → добавь `https://paparazzifashion.shop/sitemap.xml`

---

## 3. Bing Webmaster (3-й по охвату, бесплатный бонус)

Самый быстрый путь — **импорт из Google**:

1. Открой **https://www.bing.com/webmasters**
2. **Sign in** → выбери **«Import from Google Search Console»**
3. Дай разрешение → Bing подтянет твою верификацию и sitemap автоматически
4. Готово.

**Yahoo** использует Bing-индекс, так что отдельно регистрироваться не нужно.

---

## 4. Что уже работает автоматически (без твоего участия)

- ✅ **IndexNow** пингует Bing, Yandex, Seznam, Naver каждый день в 03:00 UTC при изменении каталога (см. workflow `.github/workflows/sync.yml`)
- ✅ **Sitemap** обновляется ежедневно с актуальными товарами
- ✅ **Product JSON-LD** на каждой странице → шанс на rich results (карусель фото в Google)
- ✅ **Удалённые с источника товары** автоматически уходят из sitemap, картинки удаляются, страницы товаров удаляются

---

## 5. Когда ждать первых посещений из поиска?

| Поисковик | Первая индексация | Стабильный трафик |
|-----------|-------------------|-------------------|
| Google    | 3–14 дней         | 4–8 недель        |
| Yandex    | 1–7 дней          | 2–6 недель        |
| Bing      | 1–3 дня (через IndexNow) | 2–4 недели |

Если через **3 недели** в Google Search Console будет 0 индексированных страниц — пиши, проверим что не так.
